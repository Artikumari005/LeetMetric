🚀 LeetCode Stats Tracker

A simple and interactive web application that fetches and displays LeetCode user statistics using HTML, CSS, and JavaScript.
📌 Features

    🎯 Enter a LeetCode username to track progress

    📊 Displays solved question counts by difficulty (Easy / Medium / Hard)

    📈 Animated progress circles for each category

    📋 Shows total submission statistics

    ⚠️ Validates usernames before fetching

    🧠 Uses GraphQL API and handles fetch errors

    🎨 Fully responsive design

    🛠 Tech Stack

    HTML5 – Structure

    CSS3 – Styling and animations

    JavaScript (Vanilla) – Logic, fetch API, and DOM manipulation

    LeetCode GraphQL API – Data source
